# 音频自动审核（粤语版前端版）使用说明

作者：海瀚

适用目录：
`粤语版_前端版`

---

## 1. 这个工具是做什么的

这个工具用来审核粤语配音音频。

它可以帮你完成三件事：

1. 检查音频文件、Excel 表格、文件名格式是否正常
2. 自动把拆分的 Excel 预整理后再匹配音频
3. 调用 Qwen 模型做音频审核，并导出 Excel 审核报告

建议使用顺序：

1. 先点“检查并预整理”
2. 检查通过后，再点“开始审核”

---

## 2. 需要准备什么

开始前，请先准备好以下内容：

1. 一台 Windows 电脑
2. 已安装 Python 3.10 / 3.11 / 3.12
3. 一份待审核的音频文件夹
4. 阿里云百炼（Model Studio）的 API Key
5. 本项目文件夹

---

## 3. 第一次使用：安装环境

### 第一步：确认电脑已安装 Python

按键盘 `Win + R`

输入：

```powershell
cmd
```

回车后，在黑窗口里输入：

```powershell
python --version
```

如果能看到类似：

```powershell
Python 3.12.3
```

说明 Python 已安装。

如果提示找不到 `python`，请先让技术同事帮你安装 Python。

---

### 第二步：安装需要的库

打开命令行后，进入项目目录：

```powershell
cd 你的项目路径\粤语版_前端版
```

然后执行：

```powershell
python -m pip install streamlit dashscope openpyxl
```

如果安装很慢，可以改用清华源：

```powershell
python -m pip install streamlit dashscope openpyxl -i https://pypi.tuna.tsinghua.edu.cn/simple
```

安装完成后就可以使用了。

---

## 4. 怎么获取 Qwen API Key

本工具现在使用的是阿里云百炼（Model Studio）的 Qwen API Key。

官方说明：

- [如何获取 API Key（阿里云百炼官方）](https://help.aliyun.com/zh/model-studio/get-api-key)

### 获取步骤

1. 打开阿里云百炼控制台
2. 登录你的阿里云账号
3. 在右上角切换到正确地域
4. 进入 `工作台`
5. 左侧找到 `API Key`
6. 点击 `创建 API Key`
7. 创建完成后，复制这串以 `sk-` 开头的 Key

注意：

1. 这个 Key 很重要，不要发到群里
2. 丢了就重新创建，不建议到处传播
3. 本工具使用的是百炼 API Key，不是别的平台 Key

## 5. 怎么启动前端

最简单的方法：

直接双击这个文件：

`启动前端.bat`

双击后会自动打开网页界面。

如果没有自动打开浏览器，可以手动复制窗口里显示的本地地址，一般是：

```powershell
http://localhost:8501
```

---

## 6. 前端怎么用

### 第一步：填写基础信息

打开前端后，先填写：

1. `音频目录`
2. `API Key`
3. `模型`

模型一般保持默认即可：

```text
qwen3.5-omni-plus
```

说明：

1. `音频目录` 就是待审核文件夹的路径
2. 可以直接复制粘贴路径
3. 也可以点右边的 `浏览` 按钮选择文件夹

---

### 第二步：检查并预整理

点击：

`开始检查`

系统会自动做这些事情：

1. 扫描音频文件
2. 查找 Excel
3. 自动整理拆分的 Excel
4. 干跑匹配
5. 输出检查预览

如果检查通过，再进入下一步。

如果检查不通过，请先看页面里的提示，再处理问题。

---

### 第三步：开始审核

点击：

`开始审核`

这时系统会：

1. 按条审核音频
2. 显示审核进度
3. 显示审核日志
4. 审核结束后给出报告下载按钮

---

## 7. 审核报告在哪里

审核完成后，前端页面会显示：

1. 报告文件路径
2. `下载审核报告` 按钮

报告默认保存在项目目录下的：

`reports`

---

## 8. 常见问题

### 1）点“浏览”没反应

请确认没有被别的窗口挡住。

有时候文件夹选择框会在后面，可以按 `Alt + Tab` 切换看看。

---

### 2）检查不通过

常见原因：

1. 没找到 Excel
2. 文件名格式不对
3. 音频和 Excel 内容对不上
4. 目录选错了

建议先确认：

1. 你选的是正确的总文件夹
2. 文件夹里有音频和 Excel
3. Excel 不是空的

---

### 3）API Key 填了还是不能审核

请检查：

1. API Key 是否复制完整
2. 是否以 `sk-` 开头
3. 阿里云百炼账号是否可正常使用
4. 网络是否正常

---

### 4）日志看起来乱码

如果前端日志里偶尔有乱码，通常不影响最终审核。

请优先看：

1. 是否有进度推进
2. 是否最终生成报告

---

### 5）关闭页面后还有进程

建议不要直接关浏览器标签页。

请关闭启动前端时弹出的那个黑色窗口，程序会一起结束。

---

## 9. 如果要修改提示词

前端左侧有：

`审核提示词`

你可以：

1. 直接修改
2. 点击“保存当前提示词”
3. 点击“恢复默认提示词”

默认提示词文件位置：

`review_criteria.md`

---

## 10. 给同事的最简版操作

如果你只想最快上手，照下面做：

1. 双击 `启动前端.bat`
2. 填 `API Key`
3. 选待审核文件夹
4. 点 `开始检查`
5. 检查通过后点 `开始审核`
6. 审核结束后点 `下载审核报告`
