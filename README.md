# 音频自动审核（粤语版前端版）

作者：海瀚

## 项目简介

这是一个面向粤语配音审核场景的本地工具。

它可以完成以下工作：

1. 扫描音频文件
2. 自动读取并预整理拆分的 Excel 配音表
3. 将音频与 Excel 台词进行匹配
4. 调用 Qwen 模型进行音频审核
5. 生成 Excel 审核报告
6. 提供本地前端界面，方便非技术同事使用

---

## 主要功能

### 1. 检查并预整理

在正式审核前，先进行：

1. 音频扫描
2. Excel 检查
3. 拆分表自动汇总
4. 干跑匹配
5. 风险提示

这样可以尽量避免直接浪费 API 资源。

### 2. 音频审核

审核维度以当前提示词为准，支持：

1. 内容匹配
2. 粤语发音检查
3. 语调节奏检查

### 3. 前端使用

提供 Streamlit 前端，支持：

1. 输入或浏览选择文件夹
2. 保存 API Key
3. 修改提示词
4. 查看检查进度
5. 查看审核进度
6. 下载审核报告

---

## 技术栈

本项目当前主要技术栈如下：

1. Python
2. Streamlit
3. DashScope（Qwen API）
4. OpenPyXL
5. Tkinter（本地文件夹选择）

## 目录说明

主要文件如下：

1. `app.py`
   前端页面入口

2. `launch_ui.py`
   前端启动器

3. `启动前端.bat`
   一键启动前端

4. `preflight.py`
   检查与预整理模块

5. `main.py`
   主审核流程入口

6. `scanner.py`
   音频扫描与文件名解析

7. `matcher.py`
   Excel 预整理与匹配逻辑

8. `review_criteria.md`
   审核提示词

9. `reports/`
   审核报告输出目录

---

## 启动方式

最简单的启动方式：

直接双击：

`启动前端.bat`

也可以命令行启动：

```powershell
python launch_ui.py
```
